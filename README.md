# OmlicxOS-Core

OmlicxOS-Core is the foundation of **OmlicxOS**, a custom mobile operating system for the **Google Pixel 3a**.  
Built on a Linux base with Halium integration, this core includes everything needed to run the OS: drivers, kernel configs, system files, apps, and scripts.  

This project is structured to allow developers to build, modify, and flash OmlicxOS onto Pixel 3a devices with ease.

---

## Features

- **Pixel 3a Hardware Support**: Audio, camera, display, GPS, sensors, Wi-Fi, Bluetooth drivers pre-organized.  
- **Kernel & Modules**: Customizable kernel with device-specific configuration.  
- **System Utilities**: Core binaries, libraries, and configuration files ready to use.  
- **Apps**: Minimal placeholders for launcher, messaging, dialer, workouts, and essential tools.  
- **Build Scripts**: Automated scripts for kernel build, flashing, and environment setup.  
- **Configuration**: Device tree, Halium settings, and OS config YAML.
